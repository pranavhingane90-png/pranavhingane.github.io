Featured Basic Projects

1. Smart Street Light System

Uses LDR and motion sensor to automatically control street lights. Saves power by turning lights ON only when motion is detected at night.

2. Temperature Monitoring System

Uses temperature sensor and microcontroller to monitor room temperature. Can display data on LCD and trigger buzzer if temperature exceeds limit.

3. Soil Moisture Based Irrigation

Uses soil moisture sensor to detect dryness in soil. Automatically turns ON water pump when soil becomes dry.

4. Gas Leakage Detection

Uses gas sensor to detect LPG leakage. Activates buzzer and alert system when gas concentration increases.
